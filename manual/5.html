Instalar el software

1.En la barra de notificaciones pulse la descarga completa para comenzar la instalación
2.Pulse Instalar cuando se le pregunte.
3.Pulse Siguiente para instalar el software, esto puede tomar un tiempo.
4.Pulse Abrir cuando el software haya sido instalado*

IMPORTANTE

Si no puede pulsar Abrir, significa que el software todavía está instalado en el dispositivo y debe ser desintalado primero.
Vaya a Ajustes > Administrador de Aplicaciones > SystemUpdate y luego pulse Desinstalar.
Una vez hecho esto, puede iniciar la instalación otra vez desde la página 5.

  • Si todo lo antes mencionado se cumple por favor vaya a Paso 6.
